# Base Layout Project

## Before you begin

1. Install `nodejs`, `npm`
2. `npm install npm@latest -g`
3. `apt install nodejs-legacy`

## Create project

```bash
git clone git@git.32server.in.ua:layout/patifonV2.0.git
```

## Install dependencies:

### <a name="h3-npm">**npm**</a>:

```bash
npm i
```

### <a name="h3-npm">**composer**</a>

**Install basic system**

```bash
composer install
```
